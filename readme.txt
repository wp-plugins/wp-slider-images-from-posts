Very simple plugin that allows users to display photos with the caption, in the sidebar widget

== Description ==

This plugin displays images with captions selected by you from your posts and slide them with beautiful visual effect after a time interval, which you set in settings. To setup the widget bar: select photos, sign them and specify the time interval of turning.

== Installation ==

1. Upload the ZIP content to `/wp-content/plugins/` directory or install plugin using dashboard
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add a Img-slider widget to your theme sidebar in Appearance -> Widgets menu Img-slider, select the desired images, and save them with captions.
